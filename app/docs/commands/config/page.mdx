# cxgrd config

The `config` command is used to manage cxgrd configurations

## Usage 

```bash
cxgrd config --option
```

## Example Usage 

```bash
PS C:\Users\name\project1> cxgrd config
Telemetry: enabled
  cxgrd config --disable-telemetry
  cxgrd config --enable-telemetry
```

## Options 

`--disable-telemetry` : Opt out of anonymous usage stats. No usage data will be sent.

`--enable-telemetry` : Re-enable anonymous usage stats

> Note: Anonymous usage stats help improve cxgrd. Use `--disable-telemetry` to opt out anytime
